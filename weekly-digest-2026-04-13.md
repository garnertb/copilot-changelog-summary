### 🚀 GitHub Changelog Highlights

This week opens with a headline feature: **Copilot CLI goes remote**, letting you monitor and steer a running CLI session from your phone or any web browser. It is the first time the CLI has broken out of the local terminal, and it signals a broader shift toward agent sessions you can hand off and supervise from anywhere.

#### New Features

- **Remote control CLI sessions on web and mobile** (Apr 13) - The Copilot CLI can now be supervised and steered remotely from GitHub.com or the GitHub Mobile apps (Android/iOS). When you start a session with copilot --remote (or enable it mid-session with /remote), the CLI streams activity to GitHub in real time and displays a shareable link and QR code.
